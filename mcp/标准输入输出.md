
## 工具代码实现

```python
from mcp.server.fastmcp import FastMCP

mcp = FastMCP("stdio-tools-demo")

@mcp.tool()
def add(a: int, b: int) -> int:
    """返回两个整数的和。"""
    return a + b

@mcp.tool()
def echo(text: str, uppercase: bool = False) -> str:
    """回显输入文本，可选转为大写。"""
    return text.upper() if uppercase else text

@mcp.tool()
def greet(name: str) -> str:
    """返回一条简单问候语。"""
    return f"你好，{name}！"

if __name__ == "__main__":
    mcp.run()

```

## 资源代码实现

```python
from datetime import datetime
from zoneinfo import ZoneInfo

from mcp.server.fastmcp import FastMCP

mcp = FastMCP("stdio-resources-demo", log_level="WARNING")

@mcp.resource(
    "memo://overview",
    name="overview",
    title="项目说明",
    description="返回一个固定的文本资源。",
    mime_type="text/markdown",
)
def overview() -> str:
    """返回固定的项目说明。"""
    return """# stdio resource demo

- transport: stdio
- capability: resources
- server: FastMCP
"""

@mcp.resource(
    "memo://status/today",
    name="today_status",
    title="今日状态",
    description="返回一个固定 URI 的 JSON 资源。",
    mime_type="application/json",
)
def today_status() -> dict[str, str]:
    """返回服务端当前状态。"""
    now = datetime.now(ZoneInfo("Asia/Shanghai")).isoformat(timespec="seconds")
    return {
        "status": "ok",
        "updated_at": now,
        "transport": "stdio",
        "capability": "resources",
    }

if __name__ == "__main__":
    mcp.run()
```

## 附录

• 我已经帮你把这个 MCP 加到本机 Codex 里了，名字是 tmux-demo。

  现在它的状态是：

  - codex mcp list 能看到 tmux-demo
  - codex mcp get tmux-demo 显示它是 stdio 类型
  - 配置已经写进 /home/yuchao/.codex/config.toml:138

  你这次对应的添加命令就是：

  codex mcp add tmux-demo -- /home/yuchao/AI_ask/tmux/.venv/bin/python /home/yuchao/AI_ask/tmux/server.py

  等价的手工配置写法是：

  [mcp_servers.tmux-demo]
  command = "/home/yuchao/AI_ask/tmux/.venv/bin/python"
  args = ["/home/yuchao/AI_ask/tmux/server.py"]

  这段现在已经在 /home/yuchao/.codex/config.toml:138 里了。

  你接下来这样用就行：

  1. 如果你当前这个 codex 会话是之前就打开的，先退出再重开一个 codex
  2. 进入后输入 /mcp，可以看到已启用的 MCP server
  3. 然后你就可以直接让 Codex 调用这个 server 暴露的 tools

  常用命令：

  codex mcp list
  codex mcp get tmux-demo
  codex mcp remove tmux-demo

  官方文档也确认了两种接入方式：

  - 用 CLI：codex mcp add ...
  - 或直接编辑 ~/.codex/config.toml
